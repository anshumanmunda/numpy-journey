## 📓 NumPy Notes

### 1. NumPy Fundamentals
**File:** [`01-numpy-fundamentals.ipynb`](./numpy-notes/01-numpy-fundamentals.ipynb)  
This notebook covers the essentials of NumPy, including array creation, data types, and basic indexing. 
It establishes the foundation for understanding how `ndarrays` differ from standard Python lists in terms of memory and speed. 
Key concepts include broadcasting, basic slicing, and the importance of vectorized operations.

### 2. Advanced NumPy
**File:** [`02-advance-numpy.ipynb`](./numpy-notes/02-advance-numpy.ipynb)  
Building on the basics, this section dives into complex array transformations, stacking, and splitting techniques. 
It explores universal functions (ufuncs) in depth and introduces linear algebra operations such as matrix multiplication and decompositions. 
This is where the true power of NumPy for high-level mathematical modeling is showcased.

### 3. NumPy Tricks
**File:** [`03-numpy-tricks.ipynb`](./numpy-notes/03-numpy-tricks.ipynb)  
A collection of "pro-tips" and optimization techniques to write cleaner, faster code. 
This notebook covers boolean masking, fancy indexing, and hidden gems within the library that simplify data cleaning. 
It focuses on the "NumPy-ic" way of solving problems to avoid slow Python loops.
